# deepfake-av-fusion-detector
Dual-branch deepfake detection robust to real-world video degradation. Combines ResNet-50 spatial artifact analysis with SyncNet audio-visual lip-sync scoring via a Quality-Aware Fusion Gate. Achieves 100% accuracy &amp; AUROC 1.0 on degraded LAV-DF videos. Accepted — ICCCNET 2026, Springer.
